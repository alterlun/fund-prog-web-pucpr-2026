Sobre o Projeto

Este site foi desenvolvido como parte de uma avaliação de Front-end, utilizando apenas HTML, CSS e Javascript.

O objetivo é demonstrar fluência nos conceitos fundamentais dessas tecnologias, criando uma loja com navegação simples, formulário de cadastro e tratamento de dados enviados via método GET.

Autores: Pedro Lucas Lopes Balliana, Kai Carneiro Antunes dos Santos, Ileana Kaori Kuroda

Ano: 2026
